# Budget-Tracker---Python-
A Python-based personal finance management system developed in 2024 to track income, expenses, and budgeting insights. Submitted to CBSE and awarded 100%.

FUNCTION 1: helps add data to existing files or creates a new file.
FUNCTION 2: To view total or monthly expenses.
FUNCTION 3: To view total or monthly income.
FUNCTION 4: To provide financial analysis. Provides the user with the following  
                    ◦ Trend analysis: Examining data over time to identify patterns or directions.
                    ◦ Average expenditure: Total spending divided by the number of occurrences.
                    ◦ Moving average expenditure: Average spending recalculated over a rolling time frame.
FUNCTION 5: To impose budgeting over expenses.
  budgetting(): Enables users to select budgeting options based of monthly budget.
  budgetchecker(file1, monthlybudget, month, amount): Creates files and appends expenses to existing files based of the budget value the
  user wishes to use and the month of expense
FUNCTION 6: Allows users to view finances entered in a
particular CSV file.

In today’s environment, characterized by rising costs and persistentfinancial challenges, tracking our spending can often feel daunting. This is precisely why the PyFinance Tracker emerges as the most efficient, straightforward, and rapid solution for recording daily expenses. With its exclusive reliance on Python, users can effortlessly access Excel files without the need for cumbersome data storage systems,bypassing the complications related to databases like MySQL. Even after a demanding day at work or college, investing just five minutes to log expenditures and maintain an organized financial overview becomes not only feasible but also user-friendly for the average tech-savvy individual.
